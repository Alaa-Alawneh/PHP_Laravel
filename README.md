# PHP_Laravel
A php web application as part of the internship with ITG, currently uses plain php but will use Laravel 


# Setup steps
- create an env file using the provided .env.example as template or simply reuse it
- run these commands at the root of the project
    - `composer install`
    - `docker compose up` or to run phpmyadmin with it use `docker compose --profile dev up` 
    - `php -S localhost:8000`
